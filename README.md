# Backend Inventory Management

langkah penggunaan :
1. jalankan npm install
2. jalankan docker-compose up -d
3. lakukan import preference pada layanan masing2. import realm-export.json dalam keycloak dan import db_invenotry dalam phpmyadmin (cara termudah)

layanan port yang dapat dijalankan dalam web 
<ul>
  <li>http://localhost:8080 - dashboard admin keycloak /n</li>
  <li>http://localhost:3306 - diakses lewat cmd (mysql) /n</li>
  <li>http://localhost:5000 - cek konektivitas dalam web /n</li>
  <li>http://localhost:8081 - phpmyadmin (tersambung dgn layanan db mysql)</li>
</ul>
